# 金融产品推荐系统 - 大模型增强版

## 新增功能

本项目已成功集成大模型功能，为用户提供更加个性化和专业的金融产品推荐服务。

### 主要新增功能：

1. **更详细的用户画像**
   - 投资目标（短期/中期/长期/退休/教育/购房）
   - 投资经验（新手/有一定经验/经验丰富/专业投资者）
   - 投资金额范围（小额/中等/大额）
   - 特殊需求（ESG/税务优惠/高流动性/本金保障）

2. **大模型个性化建议**
   - 基于用户详细画像的个性化投资建议
   - 专业的风险提示和注意事项
   - 投资组合配置建议
   - 未来投资规划指导

3. **增强的推荐算法**
   - 传统推荐算法 + 大模型智能分析的混合推荐
   - 更精准的个性化匹配

## 配置说明

### 环境变量配置

复制 `.env.example` 文件并重命名为 `.env`，然后填入您的大模型API密钥：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填入API密钥：
```
OPENAI_API_KEY=your_openai_api_key_here
LLM_MODEL=gpt-3.5-turbo
```

### 依赖安装

```bash
pip install -r requirements.txt
```

## 使用方法

### Web界面使用
```bash
python app.py
```
访问 http://127.0.0.1:5000

### 控制台使用
```bash
python main.py
```

## 功能特点

1. **Web界面**：
   - 新增了详细用户信息输入字段
   - 新增大模型个性化建议显示区域
   - 支持多种推荐策略选择

2. **控制台界面**：
   - 新增详细用户信息收集流程
   - 新增大模型推荐选项（第3个选项）
   - 更丰富的个性化建议展示

3. **大模型功能**：
   - 如果没有API密钥，系统会使用模拟响应
   - 支持多种大模型提供商
   - 提供专业的金融投资建议
